Full Diagnostic Suite

Everything you need
to diagnose your Pi

Run individual tests or a complete system diagnostic with a modern GUI and multiple export options.

Bootloader Recovery

Detects and repairs corrupt EEPROM, damaged bootloaders, and unreadable partitions — before Linux even loads.

Failsafe Framebuffer Mode

Launches a minimal diagnostic mode with SD card checks, power validation, and clear error guidance when Linux can’t boot.

CPU Stress Testing

Comprehensive CPU stress and temperature testing to identify thermal issues, throttling, and processor stability under load.

RAM Integrity Testing

Programmable RAM diagnostics with configurable chunk sizes and pass counts. Writes deterministic patterns and verifies readback.

SD Card Benchmarking

Full read/write speed benchmarks, health checks, and partition integrity verification. Know exactly how your storage performs.

Network Diagnostics

Ethernet and Wi-Fi testing, signal strength analysis, and network performance measurement for all configurations.

Three-Stage Architecture

How Apple Pi Diagnostics Works

A layered approach ensures your Pi can be diagnosed and repaired at any level — from bootloader corruption to full system testing.

1
Stage 1

Bootloader Recovery

Repairs critical boot components automatically. Detects and fixes EEPROM corruption, damaged bootloaders, and misconfigured settings before Linux loads.

  • EEPROM validation
  • Boot config fixes
  • Partition recovery
  • Failsafe fallback
2
Stage 2

Failsafe Mode

Lightweight framebuffer UI that works even when Linux can’t boot. Essential diagnostics without requiring a desktop environment.

  • Hardware probing
  • SD integrity checks
  • Voltage validation
  • Error explanations
3
Stage 3

Full Diagnostics GUI

A polished desktop environment powered by Python. Access the complete diagnostic suite with a modern UI when the system boots normally.

  • Individual test selection
  • Real-time monitoring
  • Reporting
  • Export options
